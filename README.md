# ХПК: Посещаемость

Веб-приложение для учета посещаемости студентов.

Разработано в рамках курсовой работы по ПМ 01 «Разработка модулей программного обеспечения для компьютерных систем» студенткой группы ИС(ТП)-31 Панагушиной М.В.

## Функциональные возможности

**Для преподавателя:**
- просмотр расписания на текущий день;
- отметка посещаемости студентов в журнале (статусы: присутствовал, отсутствовал, опоздал, уважительная причина);
- указание минут опоздания и причины отсутствия;
- массовая отметка всех присутствующих одной кнопкой;
- просмотр статистики посещаемости группы (круговая диаграмма);
- экспорт отчетов в формате Excel;
- генерация QR-кода для быстрой отметки студентов.

**Для студента:**
- просмотр своей посещаемости;
- просмотр процента посещаемости;
- отметка по QR-коду.

**Для администратора:**
- управление пользователями;
- управление группами;
- управление студентами (добавление, загрузка фото);
- управление дисциплинами;
- формирование расписания.

## Установка и запуск


1. Клонировать репозиторий:

   git clone [https://github.com/ваш_логин/khpk-attendance.git](https://github.com/marinus7772/attendance-system)
   cd khpk-attendance

2. Создать виртуальное окружение:

   python -m venv venv

3. Активировать виртуальное окружение:

   Windows:
   venv\Scripts\activate

   macOS/Linux:
   source venv/bin/activate

4. Установить зависимости:

   pip install -r requirements.txt

5. Выполнить миграции базы данных:

   python manage.py makemigrations
   python manage.py migrate

6. Создать суперпользователя (администратора):

   python manage.py createsuperuser

7. Запустить сервер разработки:

   python manage.py runserver

8. Открыть в браузере: http://127.0.0.1:8000

## Создание файла зависимостей

Если файла requirements.txt нет в репозитории, создайте его командой:

pip freeze > requirements.txt
